"> 去除Pelican代码块中的

在使用Pelican生成博客时，代码块中经常会出现红色边框，查看源文件发现红色边框是被添加了class="err"导致，于是找到主题的css文件中，去除了以下代码，红框就再也不会出现了。

.highlight .err { border: 1px solid #FF0000 } / Error /
