ubuntu tty 中文支持

Date 2014-07-22 (星期二) Tags ubuntu / tty

tty 中文乱码，解决方案：安装 zhcon 或者 fbterm。只介绍fbterm。

  1. 安装 fbterm

    $ sudo apt-get install fbterm

  2. 问题 "cann't open frame buffer device!"

    $ sudo usermod -aG video username

  3. 问题"[input] can't change kernel keymap table, all shortcuts will NOT work, see SECURITY NOTES section of man page for solution."

    $ sudo chmod u+s /usr/bin/fbterm

  4. 启动fbterm

    $ fbterm

  5. 关于修改字体大小

    $ vi .fbtermrc
font-size=16
将字体修改成合适大小就可以了。

font-names=mono
修改字体

$ fc-list
这个命令可以查看安装的字体
